Prework Study Guide

HTML

  • html is made up of elements working together to provide content and framework for all the webpages.
  • Element used to add content: text element, image element, header element, list element.
  • Elements used to divide and describe a page: Meta-related elements, Content division elements, Semantic elements/
  • The head element contains information about the webpage.
  • The body element represents the vidible content shown to the user.
  • the header element is positioned before the main element, inside the body element.

CSS

  • Cascading Style Sheet or CSS
  • CSS is the language used to style an HTML document.
  • CSS describes how HTML elements should be displayed.
  • CSS is defined by the style element.

Git

  • Git is a free and open source distributed version control system designed to handle everything from small to very large projects with speed and efficiency.
  • GIT stand for Global Information Tracker
  • It's a DevOps tool used for sourse code management.
  • It is used to track changes in the source code, enabling multiple developers to work together.
  • Some useful Git commands:
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • JavaScript is a programming language allowing the implementation of complexe features on a webpage.
  • Programmers use JavaScript to create dynamic and interactive web content such as applications and browsers.
  • Examples of JavaScript uses: Show/hide more data or user information using with the click of a button; change the color of a button after hovering the mouse hovers over it; slide by a carousel of images on the home webpage.
